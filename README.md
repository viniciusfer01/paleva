# 🇺🇸 English

## How to run the project
Clone the repository, then:
```
  cd /path/to/project
  bundle install
  rails db:create
  rails db:migrate
  rails db:seed
  rails server
```

## How to run the tests
```
  cd /path/to/project
  bundle exec rspec
```
